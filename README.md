# WhatsApp Order Confirmation System 📦

Automatically confirm COD orders via WhatsApp. 
Track customer replies in real time. 
Reduce fake orders by up to 70%.

🔗 Live Demo: https://0rder-flow-0.netlify.app

## What This System Does

- Sends automatic WhatsApp messages after every order
- Customer confirms, holds, or edits via WhatsApp buttons
- Live dashboard updates in real time
- Updates Google Sheets automatically

## Tech Stack

- Python + Flask
- Evolution API + Docker
- PostgreSQL
- Google Sheets API
- HTML/CSS Dashboard
